# hansmeru Links

A lightweight, responsive personal profile page built for GitHub Pages. It is plain HTML and CSS, so it does not need a build step.

## Edit Profile

Update the name, description, and avatar URL in `index.html`.

## Publish On GitHub Pages

1. Push these files to a public GitHub repository.
2. Open the repository settings.
3. Go to **Pages**.
4. Set the source to the default branch and `/root`.
5. Set the custom domain to `hansmeru.dev`.
6. Save and wait for GitHub to publish the site.

The `CNAME` file keeps `hansmeru.dev` attached to this GitHub Pages site.

## Files

- `index.html` is the page structure.
- `styles.css` controls the responsive design.
- `assets/` contains the simple visual assets.
